## Types d'utilisateurs

  <details>
    <summary>Profils clients</summary>

      <blockquote>
        <details>
          <summary>Pourquoi aurais-je besoin d’un profil client ?</summary>
          <blockquote>
          La création d'un compte client vous permet de contacter les prestataires de la plateforme au sujet de leurs services.
          </blockquote>
          </details>
      </blockquote>

      <blockquote>
        <details>
          <summary>Comment créer un profil client ?</summary>
          <blockquote>
            Cliquez sur le bouton de menu dans le coin supérieur droit, cliquez sur *Créer un compte client* et fournissez les informations demandées.
          </blockquote>
        </details>
      </blockquote>

      <blockquote>
        <details>
          <summary>Qui peut voir les informations de mes clients ?</summary>
          <blockquote>
            Actuellement, les informations client ne sont partagées avec personne. À l'avenir, ils pourront être partagés avec les fournisseurs que vous choisissez de contacter via la plateforme.
          </blockquote>
        </details>
      </blockquote>

      <blockquote>
        <details>
          <summary>Pourquoi as-tu besoin ___ ?</summary>
          <blockquote>
            Plus d'informations sur les raisons pour lesquelles nous collectons les informations que nous collectons sont disponibles
            sur le formulaire d'inscription. Il est également résumé ci-dessous : 
            - Nom : Nous avons besoin votre nom pour vous assurer que vous êtes une personne réelle et que, dans certains cas, vous en aurez peut-être besoin pour traiter les paiements 
            - Nom d'affichage : Nous connaissons dans certaines circonstances votre
            le nom légal peut ne pas refléter fidèlement vos intentions sur la plateforme (par exemple
            exemple si vous achetez des services pour le compte d'une entreprise), nous autorisons donc
            vous pouvez choisir un nom d'affichage si vous pensez que cela serait plus approprié. 
            - Numéro de téléphone : Nous utilisons le numéro de téléphone comme autre mesure de sécurité (en
            ajout au nom légal) et à l'avenir, nous pourrons le partager avec le service
            les prestataires que vous contactez sur la plateforme afin qu'ils puissent vous contacter
            concernant votre (vos) demande(s). 
            - Localisation : Nous collectons ces informations afin que
            nous pouvons vous suggérer des services qui pourraient vous intéresser et pour que ce service
            les fournisseurs peuvent déterminer s’ils fournissent les services demandés dans votre
            zone. 
            - Image de profil : cela contribue à rendre tout plus personnel. Ton
            l'image de profil peut s'afficher lorsque vous fournissez des commentaires sur le
            plateforme (comme les avis) ou lorsque vous contactez un fournisseur.
          </blockquote>
        </details>
      </blockquote>
  </details>

  <details>
     <summary>Profil du fournisseur</summary>
      <blockquote>
        <details>
          <summary>Pourquoi aurais-je besoin d’un profil de fournisseur ?</summary>
          <blockquote>
          L'inscription à un profil de fournisseur vous permet de proposer vos services sur la plateforme.
          </blockquote>
        </details>

        <details>
          <summary>Comment créer un profil de fournisseur ?</summary>
          <blockquote>
          Cliquez sur le bouton de menu dans le coin supérieur droit et cliquez sur *Créer un fournisseur
            Profil* et fournissez les informations demandées.
          </blockquote>
        </details>

        <details>
          <summary>Qui peut voir les informations de mon fournisseur ?</summary>
          <blockquote>
          Tous les utilisateurs enregistrés peuvent visiter votre profil de fournisseur qui affiche les informations de votre profil de fournisseur telles que l'image, l'emplacement et les langues parlées.
          Votre profil comporte également des boutons liés à votre email et à votre numéro de téléphone afin que les clients potentiels puissent vous contacter.
          </blockquote>
        </details>

        <details>
          <summary>Comment modifier mon profil de fournisseur ?</summary>
          <blockquote>
          Cliquez sur le bouton de menu dans le coin supérieur droit et cliquez sur *Mon fournisseur
            Profil*, sur la page de profil, cliquez sur le bouton *Modifier le profil*, modifiez les champs nécessaires puis cliquez sur *Enregistrer le profil*.

            <strong>Veuillez noter que les modifications ne seront pas enregistrées si vous ne cliquez pas sur le bouton</strong> <em> Enregistrer le profil </em><strong>!</strong>
          </blockquote>
        </details>
      </blockquote>
    </details>
